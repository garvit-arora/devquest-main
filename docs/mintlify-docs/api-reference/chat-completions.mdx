---
title: Chat completions
description: Send OpenAI-compatible chat requests through the DevQuest gateway.
---

## Endpoint

```http
POST /v1/chat/completions
```

## Request

```bash
curl http://localhost:8000/v1/chat/completions \
  -H "Authorization: Bearer dq_live_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.6-sol",
    "messages": [
      { "role": "user", "content": "Explain this repo in three bullets." }
    ]
  }'
```

## JavaScript

```ts
const response = await fetch("http://localhost:8000/v1/chat/completions", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${process.env.DEVQUEST_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    model: "gpt-5.6-sol",
    messages: [{ role: "user", content: "Write a short changelog." }],
  }),
});

const data = await response.json();
console.log(data.choices[0].message.content);
```

## Billing

Each successful request reserves 2 credits, then settles from actual token usage. Calls under 5,000 total tokens cost 2 credits, 5,000-9,999 cost 3 credits, 10,000-14,999 cost 4 credits, and each extra 5,000 tokens adds 1 credit.

Failed provider requests release the reserved credits.
