title: Quickstart
description: Earn credits, create a DevQuest API key, and make your first model request.
---

## 1. Sign in

Open the DevQuest dashboard and sign in with GitHub:

```text
https://devquest.garvitarora.xyz/app
```

DevQuest uses your GitHub account to verify quest completion. You do not need Azure credentials or backend setup to use the platform.

## 2. Earn credits

Open the Projects page and star an approved repository. Each approved repo star gives credits after DevQuest verifies it.

Keep the star active. If the star is removed, API access is blocked on the next verification check.

You can earn more credits from referrals, approved PR bounties, sponsor campaigns, and marketplace events when available.

## 3. Create an API key

Open API Keys, select one model, and create a secret key.

Raw keys are shown once. Store the key securely before closing the dialog.

## 4. Set your key

Use an environment variable so your key does not end up in source code.

<CodeGroup>
```powershell Windows PowerShell permanent
setx DEVQUEST_API_KEY "dq_live_your_key"
```

```powershell Windows PowerShell temporary
$env:DEVQUEST_API_KEY="dq_live_your_key"
```

```bash macOS / Linux
export DEVQUEST_API_KEY="dq_live_your_key"
```
</CodeGroup>

## 5. Call the endpoint

Make your first request with the Responses API:

```bash cURL
curl https://devquest.garvitarora.xyz/v1/responses \
  -H "Authorization: Bearer $DEVQUEST_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.6-sol",
    "input": "Say hello from DevQuest AI in one short sentence."
  }'
```

Use the [endpoint call guide](/api-reference/endpoint-call) for JavaScript, Python, chat completions, and streaming examples.

## 6. Track usage

Open Usage or Credit Wallet in the dashboard to see requests, tokens, credits spent, model usage, and API-key activity.

Each successful request costs 2 credits maximum, even when the prompt or response is large.

Failed provider requests release the reserved credits.

## 7. Use Codex

Use the `Codex setup` guide to open `C:\Users\<USERNAME>\.codex\config.toml` with Notepad, paste the DevQuest provider, save `DEVQUEST_API_KEY` permanently with `setx`, restart VS Code or Cursor, and launch Codex.
