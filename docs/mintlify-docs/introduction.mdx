DevQuest AI is an OpenAI-compatible gateway for developer-earned model access.

Sign in with GitHub, complete approved quests, earn credits, create an API key, and call DevQuest from your apps, automations, or Codex setup.

## Credit rules

- Starring one approved repository grants 500 credits.
- Every successful model request reserves 2 credits.
- Large token usage is settled after the response: 5,000 tokens costs 3 credits, 10,000 costs 4 credits, and each extra 5,000 tokens adds 1 credit.
- If a user unstars the required repository, gateway access is blocked.

## Model rules

- Each API key can use one selected model.
- Available models are shown in the dashboard when creating or editing an API key.
