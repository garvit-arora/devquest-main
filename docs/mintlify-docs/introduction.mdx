DevQuest AI is an OpenAI-compatible gateway for Azure model deployments.

Users sign in with GitHub, star approved repositories, receive credits, create an API key, and send requests through your backend. Azure provider keys and deployment names stay private on the server.

## Credit rules

- Starring one approved repository grants 500 credits.
- Every successful model request reserves 2 credits.
- Large token usage is settled after the response: 5,000 tokens costs 3 credits, 10,000 costs 4 credits, and each extra 5,000 tokens adds 1 credit.
- If a user unstars the required repository, gateway access is blocked.

## Model rules

- The backend can know about roughly 10 Azure deployments.
- Users can enable at most 3 model aliases on a single API key.
- The public model list is controlled by `DEVQUEST_PUBLIC_MODELS`.
