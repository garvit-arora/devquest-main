DevQuest lets each API key use one selected model. Choose the model when creating the key in the dashboard.

## Available models

The current dashboard model options are:

- `DeepSeek-V4-Pro`
- `gpt-5.5`
- `gpt-5.6-luna`
- `gpt-5.6-sol`

Your API key can only call the model selected for that key.

## List models

You can also fetch available models from the API:

```bash
curl https://devquest.garvitarora.xyz/v1/models
```

## Billing

Every successful model request costs 2 credits maximum. Failed provider requests release the reserved credits.
